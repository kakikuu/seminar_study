# ファイルをアップロードするまでのコード

### コード解説

#### ファイルの役割
`app.js`
- サーバーを立てる
- HTMLファイルを表示する
- アップロードされたファイルを受け取る
- アップロードされたファイルを保存する

`index.html`
- ファイルをアップロードするフォームの作成
    - app.jsで受け取ってくれる処理をしてくれる場所(エンドポイント)にアップロードするように設定
